🧠 Project Chimera — AI Security Research Orchestrator

Project Chimera is a full-stack AI security research assistant built with Flask and Google Gemini. It features persistent project memory, AI-driven code analysis, automated tool orchestration via Python subprocess, real-time process streaming using SSE, and resilient API key rotation. Chimera bridges LLM intelligence with local system execution, enabling context-aware technical reasoning, iterative debugging, and structured AI-assisted research workflows. The system includes a project memory viewer, file-assisted prompting, and robust error recovery, demonstrating advanced AI application architecture beyond traditional chatbot systems.

1. AI-Powered Technical Analysis Engine

Chimera uses Google Gemini models with custom system instructions to act as:

  • A code reviewer
  • A logic flaw detector
  • A system design analyst
  • A technical strategy assistant

It can:

  • ✔ Analyze Python proof-of-concept code
  • ✔ Break down how a script works step-by-step
  • ✔ Identify weaknesses, edge cases, and logic gaps
  • ✔ Suggest structural improvements
  • ✔ Explain complex technical systems in plain language

This makes Chimera function like an AI senior engineer/security analyst.

2. Project-Based Persistent Memory System

Unlike normal chatbots, Chimera has long-term memory per project.

How it works:

  • Each project gets its own JSON memory file
  • Every user input + AI response is stored
  • Conversations can be resumed days later
  • Context builds over time

This enables:

  • 🧩 Multi-session research continuity
  • 🧠 Learning from past attempts
  • 📂 Organized project intelligence
  • 🔁 Iterative debugging and refinement

You built a stateful AI system, not just stateless prompts.

3. API Key Rotation & Fault Tolerance

Chimera is designed for production-style resilience.

If an API key hits limits (429 errors), the system:

  • Automatically rotates to the next key
  • Retries on temporary server failures
  • Handles service outages gracefully

This provides:

  • ⚙️ High availability
  • 📈 Scalability
  • 🛡️ Robust error handling

Most hobby AI projects don’t include this — this is engineering-level architecture.

4. POC & Context-Aware AI Reasoning

Chimera doesn't just take text prompts. It combines:

  • Problem description
  • Recon/context files
  • Full source code (POC uploads)

It feeds structured multi-part context to the AI model, enabling:

  • 🔍 Deep code + environment analysis
  • 🧠 Multi-source reasoning
  • 📎 File-assisted AI intelligence

That’s advanced prompt architecture, not basic chat.

5. Automated Tool Orchestration Framework

Chimera can generate Python scripts that execute real tools on the user's machine.

The design includes:

  • subprocess automation
  • Tool availability checks
  • Output capture & summarization
  • One-click execution model

This makes Chimera a bridge between:

  • AI reasoning → Real system actions

That’s an agentic AI architecture, not just Q&A.

6. Live Recon / External Script Streaming

You integrated a separate reconnaissance script (“Spotter”) and stream its output to the browser using Server-Sent Events (SSE).

Capabilities include:

  • 📡 Live terminal output streaming
  • 📁 Auto-detecting generated result files
  • 🛑 Remote stop signals via flag file
  • 📤 Real-time log feed to UI

This is full process orchestration + real-time frontend streaming.

7. AI Output Parsing System

Chimera intelligently separates:

  • Explanations
  • Code blocks

Using regex parsing, it structures AI output for UI display.

This improves:

  • 🖥️ Readability
  • 📜 Code extraction
  • 📚 Documentation quality

8. Built-in Project Memory Viewer UI

You didn’t just store memory — you built a viewer interface.

Features:

  • Lists all project histories
  • Displays full conversations
  • Renders code blocks separately
  • Clean formatted UI

That’s AI observability tooling — rare in student projects.

9. Custom Safety Configuration

The system defines explicit safety settings and handles blocked responses gracefully, showing:

  • ⚖️ AI moderation awareness
  • 🧠 Controlled model behavior
  • 🔧 Adaptive prompting strategies
